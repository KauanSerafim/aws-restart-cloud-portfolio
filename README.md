Este repositório documenta minha jornada de aprendizado durante o programa **AWS re/Start**, focado em fundamentos de computação em nuvem, Linux, redes e serviços da AWS.

Ao longo do programa realizo diversos **laboratórios práticos**, onde aplico conceitos teóricos utilizando serviços da AWS e ferramentas de linha de comando.

O objetivo deste repositório é:

- Registrar minha evolução no aprendizado de **Cloud Computing**
- Documentar **laboratórios práticos realizados**
- Demonstrar habilidades técnicas em **AWS, Linux e automação**
- Servir como **portfólio técnico** para oportunidades na área de Cloud

## Sobre o programa AWS re/Start

O **AWS re/Start** é um programa de capacitação focado em preparar profissionais para carreiras em **computação em nuvem**, abordando conceitos como:

- Infraestrutura em nuvem
- Redes
- Segurança
- Automação
- Administração de sistemas Linux
